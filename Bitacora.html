PROYECTO FINAL DE FUNDAMENTOS DE SOFTWARE

Landing page

Una landing page (página de destino o de aterrizaje) es una página web independiente diseñada con un único objetivo de conversión, como generar clientes potenciales, registrar usuarios o vender un producto, para lo cual suele eliminar el menú de navegación y otros elementos para enfocar la atención del visitante en una llamada a la acción (CTA) específica.

Heurística

Una heurística es una estrategia cognitiva que un individuo pone en práctica para solucionar un problema complejo, como tomar una decisión o formarse un juicio.

Software

El software es el conjunto de programas informáticos, instrucciones, reglas y documentación que permiten a un dispositivo electrónico (como una computadora o un teléfono) realizar tareas específicas.

Software de sistema

En informática, el software de sistema, denominado también software de base, es el conjunto de instrucciones que no permiten el manejo de la computadora. Una computadora sin software de sistema se hace inmanejable.

Hardware

El hardware se refiere al conjunto de componentes físicos y tangibles de un sistema informático, como la CPU, el monitor, el teclado, el disco duro o la memoria RAM, que permiten que un dispositivo funcione.

TIC

Las TIC (Tecnologías de la Información y la Comunicación) son el conjunto de recursos, herramientas y sistemas que permiten la creación, almacenamiento, procesamiento, acceso y transmisión de información en diversos formatos, como voz, datos, texto, video e imágenes.

Drivers

Un driver (o controlador) es un tipo de software que actúa como un intérprete para que el sistema operativo (como Windows o macOS) pueda comunicarse y controlar un componente de hardware específico de la computadora, como una tarjeta de video o una impresora.

Linux

Linux es un sistema operativo de código abierto, gratuito y basado en Unix, creado por Linus Torvalds. Funciona como núcleo (kernel) y junto con otros programas forma un sistema operativo completo que puede personalizarse, compartirse y modificarse libremente.

Windows

Windows es una familia de sistemas operativos desarrollados por Microsoft que permiten el funcionamiento de computadoras personales, portátiles, tabletas y otros dispositivos, ofreciendo una interfaz gráfica fácil de usar, que permite realizar tareas como navegar por internet, usar programas de ofimática, jugar y administrar archivos.

Navegador web

Un navegador web es una aplicación o programa de software que permite a los usuarios acceder, recuperar y visualizar contenido de la World Wide Web (WWW), como páginas web, imágenes, videos y otros archivos.

Dispositivos IOT

Los dispositivos IoT (Internet de las Cosas) son objetos físicos equipados con sensores, software y otras tecnologías que les permiten conectarse a Internet y a otras redes, recopilar e intercambiar datos entre sí y con otros sistemas.

Ubuntu

Ubuntu es un sistema operativo gratuito y de código abierto, basado en Linux, desarrollado por Canonical y lanzado en 2004 para hacer de Linux un sistema más accesible.

Debian

Debian es una distribución de GNU/Linux basada en el software libre, conocida por su estabilidad, seguridad y su compromiso con la comunidad del software libre.

Fedora

Fedora es una distribución del sistema operativo Linux, desarrollada por el Proyecto Fedora, que proporciona software libre y de código abierto con un enfoque en la adopción temprana de nuevas tecnologías para usuarios y desarrolladores.

macOS

macOS es el sistema operativo propietario gráfico de Apple Inc. para sus computadoras de escritorio y portátiles, conocidas como Mac. Desarrollado sobre la arquitectura UNIX y un núcleo basado en BSD, macOS ofrece una interfaz de usuario intuitiva, optimizada para el hardware de Apple, lo que resulta en un rendimiento eficiente y una fuerte integración con el ecosistema de productos de la compañía.

macOS Sequoia

macOS Sequoia (versión 15) es el sistema operativo más reciente para computadoras Mac de Apple, anunciado en WWDC 2024, que destaca por su integración con iPhone a través de iPhone Mirroring, la nueva aplicación Contraseñas para una gestión centralizada de credenciales, y las capacidades de Apple Intelligence que ofrecen ayuda contextual y escritura avanzada.

BIOS UEFI

UEFI (Interfaz de Firmware Extensible Unificada) es una actualización moderna de la BIOS (Sistema Básico de Entrada/Salida) tradicional en las computadoras.

Microsoft

Microsoft en tecnología se refiere a la compañía multinacional estadounidense que es uno de los mayores proveedores de software y servicios en la nube, hardware y videojuegos, incluyendo productos icónicos como el sistema operativo Windows, la suite de productividad Microsoft 365, el navegador Edge y las consolas de videojuegos Xbox.

Office

Microsoft Office (ahora Microsoft 365), un conjunto de aplicaciones de software de productividad desarrollado por Microsoft que incluye programas para el procesamiento de textos (Word), hojas de cálculo (Excel), presentaciones (PowerPoint), correo electrónico (Outlook) y otras tareas de ofimática.

Google

Google es una empresa y un motor de búsqueda que organiza información mundial para hacerla accesible y útil, y que se ha convertido en un verbo para buscar en Internet.

Google workspace

Google Workspace es una suite de software en la nube desarrollada por Google que ofrece herramientas de productividad y colaboración como Gmail, Google Drive, Google Meet, Docs, Sheets, y Slides, integrando correo electrónico, mensajería, colaboración en documentos y videoconferencias para facilitar el trabajo en equipo y la comunicación fluida desde cualquier lugar y dispositivo.

Proyectos UX/U

Un proyecto UX/UI en tecnología busca crear productos digitales (aplicaciones, sitios web) que sean tanto atractivos visualmente (Diseño de Interfaz de Usuario o UI) como fáciles y satisfactorios de usar (Diseño de Experiencia de Usuario o UX).

Interacción sincrónica

La interacción sincrónica en tecnología se refiere a la comunicación y el intercambio de información que ocurre en tiempo real entre dos o más sistemas o usuarios, donde todos los participantes están activos y disponibles simultáneamente para recibir y enviar datos.

Interacción asincrónica

La comunicación asincrónica es un término que se origina en las telecomunicaciones. Se refiere a la transmisión de datos sin sincronizar con un reloj, lo que significa que esta información proviene de un intercambio intermitente y no de un flujo constante.

Editor de código

Un editor de código es un software especializado diseñado para que programadores y desarrolladores escriban, editen y gestionen código fuente de programas.

Editor de texto

Un editor de texto es una aplicación informática que permite crear, abrir, ver y modificar archivos de texto sin formato (texto plano) y código fuente.

Compilador

Un compilador es un programa que traduce el código fuente escrito en un lenguaje de programación de alto nivel, como C++ o Java, a otro lenguaje, generalmente código máquina o código objeto, que la computadora puede ejecutar directamente.

IDE editor de desarrollo integrado

IDE significa Entorno de Desarrollo Integrado, que es una aplicación de software que reúne las herramientas esenciales que los programadores necesitan para desarrollar software en una única interfaz. Estas herramientas suelen incluir un editor de código, un depurador, un compilador o intérprete, y a veces simuladores, facilitando el proceso de codificación, prueba y empaquetado de software.

Framework

En tecnología, un framework es un conjunto de herramientas, código reutilizable y directrices que proporcionan una estructura base para desarrollar software, aplicaciones o sistemas.

APU

Una APU (Unidad de Procesamiento Acelerado) es un tipo de microprocesador que integra tanto una unidad central de procesamiento (CPU) como una unidad de procesamiento gráfico (GPU) en el mismo chip, una característica distintiva de algunos procesadores de AMD.

API

Una API (Interfaz de Programación de Aplicaciones) es un conjunto de reglas y protocolos que permite a dos programas de software comunicarse e intercambiar datos o funcionalidades de forma estructurada y bien documentada.

Software de desarrollo profesional

El software de desarrollo profesional en tecnología son herramientas y entornos que los programadores y desarrolladores utilizan para crear, mantener, probar y desplegar otros programas, aplicaciones y sistemas de manera eficiente y efectiva.

Software de desarrollo educativo

El software de desarrollo educativo son programas y herramientas digitales diseñados para facilitar los procesos de enseñanza y aprendizaje, que pueden incluir desde la creación de cursos y la gestión de contenidos hasta la interacción del estudiante con la materia, el desarrollo de habilidades y la colaboración entre usuarios, buscando optimizar los resultados educativos.

Software de desarrollo de ingeniería

El software de desarrollo de ingeniería en tecnología se refiere al conjunto de herramientas, técnicas y procesos para crear, diseñar, implementar y mantener sistemas de software complejos que se aplican en diversas ramas de la ingeniería y la ciencia.

Licencia freemium

Una licencia freemium en tecnología es un modelo de negocio que ofrece un producto o servicio con funcionalidades básicas gratuitas para atraer a una gran base de usuarios, y luego monetiza el servicio ofreciendo funciones avanzadas, de mayor capacidad o eliminando limitaciones a través de pagos, es decir, una versión "premium" de pago.

Licencia propietario

Una licencia propietaria, o de código cerrado, es un acuerdo legal que otorga el permiso para usar un software, pero restringe su modificación, distribución y acceso al código fuente, ya que los derechos de autor pertenecen al creador o desarrollador.

Licencia libre

Una licencia libre en tecnología, ya sea para software u otra obra creativa, es un acuerdo legal que otorga al receptor amplias libertades para usar, estudiar, modificar y redistribuir la obra.

Antivirus

Un antivirus es un software de seguridad diseñado para detectar, prevenir y eliminar software malicioso (malware) como virus, gusanos, troyanos y ransomware de computadoras y otros dispositivos electrónicos.

Firewall

Un firewall o cortafuegos es un sistema de seguridad de red que actúa como una barrera entre una red interna confiable y una red externa no confiable, como Internet.

Inkscape

En tecnología, Inkscape es un editor de gráficos vectoriales gratuito y de código abierto, similar a programas profesionales como Adobe Illustrator

Adobe Illustrator

Adobe Illustrator es una aplicación de software estándar en la industria para el diseño de gráficos vectoriales, utilizada para crear y editar logotipos, iconos, ilustraciones, tipografía y otros diseños que requieren escalabilidad sin pérdida de calidad.

Call to action

Una Call to Action (CTA) o llamada a la acción es un mensaje o elemento visual (como un botón o texto con hipervínculo) que se coloca en una plataforma digital (sitio web, aplicación, etc.) para incitar al usuario a realizar una acción específica, como suscribirse, descargar, comprar o registrarse.

Pop up

En tecnología, un pop-up, o ventana emergente, es un elemento gráfico que aparece automáticamente sobre el contenido principal de una página web para captar la atención del usuario.

Modal

En tecnología, un modal es un cuadro de diálogo o ventana emergente que aparece sobre el contenido principal de una aplicación o página web, bloqueando la interacción con la interfaz principal hasta que el usuario la cierra.

HTML

HTML, siglas de "Lenguaje de Marcado de Hipertexto" (Hypertext Markup Language), es el lenguaje estándar para estructurar y presentar contenido en la web.

Python

Python es un lenguaje de programación interpretado, de alto nivel y de código abierto, conocido por su sintaxis clara y legible, ideal para principiantes y expertos.

PSeInt

PSeInt es una herramienta educativa multiplataforma y de software libre que ayuda a los estudiantes a aprender los fundamentos de la programación utilizando un pseudo-lenguaje en español y un editor de diagramas de flujo.

AI

Inteligencia Artificial (IA) es la ciencia de crear máquinas que pueden realizar tareas que normalmente requieren inteligencia humana, como el aprendizaje, el razonamiento, la percepción y la toma de decisiones.

Redes neuronales

Una red neuronal en tecnología es un sistema de inteligencia artificial inspirado en el cerebro humano que utiliza nodos interconectados (neuronas artificiales) para procesar datos y reconocer patrones complejos.

Perceptron

Un perceptrón es un algoritmo que aprende los pesos de un límite de decisión basándose en los datos de entrenamiento, lo que le permite clasificar las muestras correctamente. Ajusta iterativamente el límite de decisión hasta que todas las muestras se clasifiquen correctamente.

NLP

El procesamiento de lenguaje natural (NLP) combina modelos de lingüística computacional, machine learning y aprendizaje profundo para procesar el lenguaje humano.

AI fuerte

La Inteligencia Artificial (IA) Fuerte, también conocida como Inteligencia Artificial General (IAG), es un sistema hipotético que posee una inteligencia, autoconciencia y capacidades cognitivas equiparables a las humanas.

AI bot

Un "IA bot" es un tipo de chat bot (bot de conversación) que utiliza la inteligencia artificial (IA) para simular conversaciones humanas de manera natural, tanto por escrito como por voz.

AI asistente

Un Asistente de IA es una aplicación de software que utiliza inteligencia artificial para realizar tareas y brindar asistencia en tiempo real, imitando la interacción humana.

AI agente

Un agente de IA en tecnología es un sistema de software que utiliza inteligencia artificial para percibir su entorno, tomar decisiones de forma autónoma y realizar acciones para alcanzar objetivos predeterminados por los humanos.

AI débil

La IA débil, también conocida como IA estrecha o Inteligencia Artificial Estrecha(IAN), se refiere a los sistemas de inteligencia artificial diseñados y entrenados para realizar una tarea específica y bien definida.

AI estrecha

La Inteligencia Artificial (IA) Estrecha, o "IA débil", es un sistema informático diseñado para realizar una tarea específica y limitada, como el reconocimiento de voz o la identificación de imágenes.

Prompt

Un Prompt o mensaje, en el contexto informático, es un comando o elemento textual presentado en un sistema operativo o aplicación para indicar que está listo para recibir datos o ejecutar instrucciones.

Parámetros

En informática, un parámetro se define como "una referencia o valor que se pasa a una función, procedimiento, subrutina, comando o programa".

Árboles de decisión

Un árbol de decisión (decision tree) es un algoritmo de aprendizaje supervisado no paramétrico, que se utiliza tanto para tareas de clasificación como de regresión. Tiene una estructura jerárquica, de árbol, que consta de un nodo raíz, ramas, nodos internos y nodos de hoja.

Ciberseguridad

La ciberseguridad es la práctica de proteger sistemas informáticos, redes, dispositivos y datos digitales de ataques, daños y accesos no autorizados, asegurando la confidencialidad, integridad y disponibilidad de la información.

Vibe coding

El vibe coding es una práctica emergente de desarrollo de software que usa Inteligencia Artificial (IA) para generar código funcional a partir de instrucciones de lenguaje natural, lo que acelera el desarrollo y hace que la creación de apps sea más accesible, especialmente para quienes tienen poca experiencia en programación.

Datos sintéticos

Los datos sintéticos son información generada artificialmente que imita las características y propiedades de los datos reales, pero no contiene información personal o sensible.

Usuarios sintéticos

Los usuarios sintéticos son entidades creadas artificialmente gracias a la IA que imita el comportamiento, las preferencias y los patrones de uso de usuarios reales.

User journey

En tecnología, un User Journey es la representación de los pasos, pensamientos y emociones de un usuario al interactuar con un producto o servicio digital para alcanzar un objetivo específico.

AI taxonomías

La taxonomía de la inteligencia artificial no solo revela el estado actual y el potencial futuro de esta tecnología, sino que también plantea preguntas fundamentales sobre cómo la humanidad se relaciona con máquinas cada vez más inteligentes y autónomas.

Machine learning

El Machine Learning (ML), o aprendizaje automático, es una rama de la inteligencia artificial (IA) que permite a los sistemas informáticos aprender de los datos para realizar tareas sin ser programados explícitamente para cada caso.

Algoritmo

En tecnología, un algoritmo es una secuencia finita y ordenada de instrucciones, pasos o reglas bien definidas que se siguen para realizar una tarea específica, resolver un problema o alcanzar un objetivo concreto.

Computer vision

La visión por computadora (Computer Vision) es una rama de la inteligencia artificial (IA) que permite a las máquinas "ver" e interpretar datos visuales, como imágenes y videos, de manera similar a como lo haría el ojo humano.

Recopilación de datos

La recopilación de datos es el proceso de recogida de información mediante la observación y la experimentación

Prompts de imagen

Un prompt es una instrucción o descripción que se le da a una IA para que genere una imagen. La calidad de ese resultado depende directamente de cómo esté redactado el texto.

Prompts de audio

Los prompts de audio son el detonante en la generación de música, efectos de sonido y diálogos de voz mediante inteligencia artificial.

Prompts de texto

Un Prompt o mensaje, en el contexto informático, es un comando o elemento textual presentado en un sistema operativo o aplicación para indicar que está listo para recibir datos o ejecutar instrucciones.

Github

GitHub es una plataforma web de alojamiento de código basada en la nube que utiliza el sistema de control de versiones Git para permitir la colaboración eficiente en proyectos de software.

Figma

Figma es una herramienta de diseño y prototipado colaborativa basada en la nube que permite a los equipos crear, probar y agilizar el desarrollo de interfaces de usuario (UI) y experiencias de usuario (UX) de forma simultánea en tiempo real, desde cualquier lugar y dispositivo con acceso a internet.

CSS

En tecnología, CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) es un lenguaje de estilo utilizado para definir la presentación visual de documentos escritos en lenguajes de marcado como HTML.

Javascript

JavaScript (JS) es un lenguaje de programación de alto nivel y basado en texto, usado para crear páginas web interactivas y dinámicas al añadir funcionalidades como animaciones, validaciones de formularios y actualizaciones de contenido en tiempo real.

Google colab

Colab es un servicio alojado de Jupyter Notebook que no requiere configuración y que ofrece acceso sin coste económico a recursos de computación, como GPUs y TPUs.

Números flotantes

En tecnología, los números flotantes o de punto flotante son una forma de representar números reales, incluyendo fracciones y valores muy grandes o muy pequeños, utilizando un formato similar a la notación científica.

input en python

En Python, input() es una función integrada que permite al programa obtener datos directamente del usuario a través del teclado, pausando la ejecución hasta que el usuario ingrese un valor y presione "Enter".

Crud

En tecnología, CRUD es un acrónimo de las cuatro operaciones básicas sobre información almacenada: Crear (Create), Leer (Read), Actualizar (Update) y Eliminar (Delete).

JPG

En tecnología, JPG (o JPEG) es un formato de archivo de imagen digital que utiliza un método de compresión con pérdida para reducir significativamente el tamaño del archivo, haciendo que las imágenes, especialmente fotografías e ilustraciones, sean más manejables para el almacenamiento y la distribución en la web o por correo electrónico.

PNG

En tecnología, PNG son las siglas de Portable Network Graphics, un formato de imagen ráster que se caracteriza por su compresión sin pérdida de calidad, su capacidad de fondos transparentes o semitransparentes, y un amplio soporte de color.

WEB

En tecnología, el término "formato Web" generalmente se refiere a las estructuras y el contenido (como HTML, CSS, JavaScript) que constituyen los elementos de un sitio o página web, permitiendo su visualización y uso a través de un navegador.

PNL

El PNL es un tipo de inteligencia artificial (IA). Los modelos modernos de PNL se construyen principalmente mediante aprendizaje automático, y también se basan en el campo de la lingüística, el estudio del significado del lenguaje.

MD

La dinámica molecular ( MD ) es un método de simulación por computadora para analizar los movimientos físicos de átomos y moléculas.

Kick off

En tecnología, "kick off" (o reunión de inicio) es la primera reunión formal de un proyecto para alinear al equipo y a todas las partes interesadas sobre los objetivos, alcance, responsabilidades, cronograma y plan de acción del proyecto.

Stakeholder

En tecnología, un stakeholder es cualquier persona, grupo u organización que tiene un interés o influencia en un proyecto tecnológico, un producto, un sistema o una empresa, ya sea porque se ve afectado por él o porque puede afectar.

Ventana de contexto AI

La ventana de contexto (o "longitud de contexto") de un modelos de lenguaje de gran tamaño (LLM) es la cantidad de texto, en token, que el modelo puede considerar o "recordar" en cualquier momento. Una ventana de contexto más grande permite que un modelo de IA procese entradas más largas e incorpore una mayor cantidad de información en cada output.

Whatsapp

WhatsApp es una aplicación de mensajería instantánea multiplataforma que permite a los usuarios enviar mensajes de texto, audio, vídeo, imágenes, documentos, realizar llamadas y videollamadas, todo utilizando la conexión a internet del dispositivo móvil.

Facebook

En tecnología, Facebook es una red social en línea fundada en 2004 que permite a los usuarios crear un perfil personal, conectar con amigos, familiares y otras personas, y compartir contenido como fotos, videos y texto.

Instagram

Instagram es una plataforma de redes sociales basada en la compartición de fotos y videos que funciona principalmente a través de una aplicación móvil.

Linkedin

En el ámbito de la tecnología, LinkedIn es una red social profesional que funciona como un ecosistema para conectar a profesionales, empresas y reclutadores, permitiendo la creación de perfiles laborales, el intercambio de información, la búsqueda de empleo, el networking, la promoción de productos y servicios, y la gestión de contactos comerciales.

Android

Android es un sistema operativo móvil de código abierto, desarrollado por Google y basado en el núcleo de Linux, que funciona en una amplia gama de dispositivos como smartphones, tablets y otros dispositivos inteligentes.

Sintaxis

La sintaxis es fundamental en los lenguajes de programación porque establece las reglas para escribir código válido.

AMD

AMD son las siglas de Advanced Micro Devices, Inc., una empresa estadounidense especializada en el diseño y fabricación de hardware de computación de alto rendimiento, incluyendo procesadores (CPU), tarjetas gráficas (GPU), sistemas en chip (SoC) y matrices de puertas programables en campo (FPGA). Sus productos, como las líneas Ryzen y Radeon, se utilizan en ordenadores personales, centros de datos, inteligencia artificial, gaming y otras aplicaciones industriales.

Intel

En tecnología, Intel es una compañía multinacional estadounidense que diseña y fabrica componentes de hardware, como procesadores (CPU), chips y otros semiconductores, para una amplia gama de dispositivos, incluyendo computadoras personales, servidores y centros de datos.

Nvidia

Nvidia diseña unidades de procesamiento de gráficos para los mercados de videojuegos y profesionales, así como sistema en unidades de chip (SoC) para el mercado de computación móvil y automotriz.

Dato

En tecnología, un dato es un elemento descontextualizado —como un número, letra o símbolo— que representa una observación, hecho o cualidad, y que una computadora puede interpretar, almacenar y procesar.

Información

En tecnología, la información es el producto procesado, organizado y significativo de los datos, que se utiliza para la toma de decisiones y la generación de conocimiento.

Single sign on (SSO)

El Single Sign On (SSO), o inicio de sesión único, es un método de autenticación que permite a los usuarios acceder a múltiples aplicaciones y sitios web con una sola combinación de credenciales (usuario y contraseña) en lugar de tener que iniciar sesión por separado en cada uno.

Tempmail

En tecnología, temp mail se refiere a un servicio de correo electrónico temporal o desechable que genera direcciones de correo con una validez corta para recibir mensajes, proteger la privacidad del usuario y evitar el spam.

Código QR

Un Código QR (Quick Response) es un tipo de código de barras bidimensional que, al ser escaneado por un smartphone o lector especializado, puede almacenar y proporcionar acceso rápido a información digital como direcciones de sitios web (URL), datos de contacto o textos.

Código de barras

En tecnología, un código de barras es una representación visual de datos mediante un patrón de barras verticales o un conjunto de cuadros (en 2D) que, al ser escaneado por un lector electrónico, codifica información alfanumérica para identificar productos o artículos.

LLM

Los modelos de lenguaje de gran tamaño, también conocidos como LLM, son modelos de aprendizaje profundo muy grandes que se pre entrenan con grandes cantidades de datos.

VPN

Una VPN (Red Privada Virtual) en tecnología es un servicio que crea un túnel de conexión cifrado y seguro entre tu dispositivo e Internet, utilizando un servidor remoto para enmascarar tu dirección IP real y proteger tu actividad en línea.

Visual studio code

Visual Studio Code, al que conocemos también como VSCode, es un editor de código para programadores gratuito, de código abierto y multiplataforma.

Pseudocódigo

El pseudocódigo es una descripción detallada y de alto nivel de los pasos que sigue un algoritmo o programa, escrita en un formato que combina lenguaje natural (como el inglés) con convenciones de programación.

Stack

Un stack tecnológico o "tech stack" que es el conjunto de herramientas, lenguajes y tecnologías (como bases de datos, frameworks y servidores) que se utilizan para desarrollar una aplicación web o un sistema digital.

JSP

(JavaServer Pages, ahora Jakarta Server Pages) es una tecnología del lado del servidor para crear páginas web dinámicas que mezclan código Java con HTML, XML u otros tipos de documentos.

TAC

TAC se refiere a las Tecnologías del Aprendizaje y del Conocimiento, un enfoque educativo que utiliza las Tecnologías de la Información y Comunicación (TIC) con un propósito pedagógico y formativo, no solo informativo, para optimizar la enseñanza y el aprendizaje.

Librerías

En tecnología, una librería (o biblioteca) es una colección de código preescrito y reutilizable que proporciona funcionalidades ya desarrolladas para resolver problemas comunes en la programación.

Romcat

NetBean

NetBeans es un Entorno de Desarrollo Integrado (IDE) gratuito y de código abierto, principalmente enfocado en el desarrollo de aplicaciones Java, pero también compatible con otros lenguajes como PHP, C/C++ y HTML5.

Wordpress

En tecnología, WordPress es un Sistema de Gestión de Contenidos (CMS) de código abierto, gratuito y muy popular, utilizado para crear y administrar sitios web y blogs.

SQL

En tecnología, SQL (Lenguaje de Consulta Estructurado) es un lenguaje de programación estándar utilizado para gestionar y manipular datos dentro de bases de datos relacionales, que organizan la información en tablas con filas y columnas.

ABM

Actividad-Based Management (Gestión basada en actividades), que busca la mejora continua de las operaciones reduciendo costos mediante el uso de datos de la actividad productiva.

Deep learning

El Deep Learning (aprendizaje profundo) es un tipo de inteligencia artificial que utiliza redes neuronales artificiales con múltiples capas para analizar grandes volúmenes de datos y aprender patrones complejos de forma autónoma.

Spotify

Spotify es un servicio de streaming de audio y vídeo de origen sueco que ofrece acceso a millones de canciones, podcasts y otros contenidos multimedia, permitiendo a los usuarios escucharlos sin necesidad de descargarlos y con la posibilidad de hacerlo de manera gratuita con publicidad o mediante una suscripción de pago (Spotify Premium) para disfrutar de funciones adicionales como la descarga de contenido y la experiencia sin interrupciones, todo ello a través de una aplicación disponible en una amplia variedad de dispositivos.

MDA

Una pantalla MDA (Monochrome Display Adapter) es un adaptador de vídeo que solo puede mostrar texto, no gráficos, y utiliza una pantalla monocromática de un solo color (como el fósforo verde).

CGA

Una pantalla CGA se refiere a la visualización de una computadora utilizando el adaptador de gráficos a color (CGA) desarrollado por IBM en 1981, que fue el primer estándar gráfico a color para el IBM PC.

EGA

Una pantalla EGA (Enhanced Graphics Adapter o Adaptador Gráfico Mejorado) es un estándar de gráficos de computadora desarrollado por IBM en 1984, que ofreció mejoras en color y resolución sobre el anterior estándar CGA y sirvió de puente tecnológico antes de ser reemplazado por el estándar VGA en 1987.

VGA

Una pantalla VGA (Video Graphics Array o Matriz de Gráficos de Video) es una interfaz analógica y un estándar de video que define la resolución de 640x480 píxeles y el conector físico de 15 pines utilizado para conectar dispositivos de video, como tarjetas gráficas, a monitores y proyectores.

CRT

Una pantalla CRT (tubo de rayos catódicos) es un dispositivo de visualización basado en un tubo de vacío que utiliza un haz de electrones para excitar el fósforo en la pantalla, creando así imágenes.

SVGA

SVGA, o Super Video Graphics Array, es un estándar de visualización que ofrece resoluciones y colores más altos que su predecesor, VGA, permitiendo una mayor claridad y detalle en la imagen.

LCD

Una pantalla LCD es una pantalla plana que usa «cristales líquidos» para crear imágenes bloqueando o permitiendo el paso de la luz, en lugar de emitirla.

Plasma

Una pantalla de plasma es una tecnología de visualización plana que usa diminutas celdas llenas de un gas ionizado (plasma) para generar imágenes.

OLED

Una pantalla OLED es un tipo de pantalla que usa diodos orgánicos emisores de luz (OLED) para producir su propia luz, píxel a píxel, eliminando la necesidad de una retroiluminación.

AMOLED

Una pantalla AMOLED (Diodo Orgánico Emisor de Luz de Matriz Activa) es una tecnología de visualización que utiliza compuestos orgánicos para producir su propia luz al recibir una corriente eléctrica.

QLED

La tecnología QLED utiliza puntos cuánticos (Quantum Dots), que son diminutas partículas semiconductoras de tamaño nanométrico, para mejorar la calidad de la imagen en los televisores.

WOLED

WOLED (White OLED) es una tecnología de pantalla que utiliza diodos emisores de luz orgánicos blancos (emiten luz blanca) y los combina con filtros de color para crear las imágenes en color en las pantallas, a diferencia de los OLED tradicionales que usan emisores RGB (rojo, verde, azul). Este enfoque permite a las pantallas WOLED lograr negros perfectos, un contraste excepcional y un brillo superior, ya que el emisor blanco no envejece de forma desigual como los emisores individuales de color.

Variable

En tecnología, una variable es un espacio de memoria que almacena un valor cambiante, como números o texto, que puede ser modificado durante la ejecución de un programa.

RGB

La tecnología RGB (por sus siglas en inglés, Rojo, Verde y Azul) es un modelo de color aditivo que combina luz de estos tres colores primarios en diversas intensidades para reproducir una amplia gama de colores.

Lúmenes

En tecnología, el término "lúmenes" se refiere al flujo luminoso o la cantidad total de luz visible que emite una fuente de luz, como una bombilla o una pantalla. Es una medida de brillo que permite comparar la luminosidad de diferentes dispositivos, de modo que un mayor número de lúmenes indica una luz más brillante.

volátil

En tecnología, "volátil" describe un componente o dato que es temporal y pierde su contenido cuando se interrumpe la alimentación de energía, como la memoria RAM.

Switch

Un switch, o conmutador, es un dispositivo de red que interconecta varios dispositivos (como computadoras, impresoras, servidores y puntos de acceso inalámbricos) dentro de una Red de Área Local (LAN) para permitirles comunicarse eficientemente.

Router

Un router es un dispositivo de red que conecta distintas redes informáticas, como tu red doméstica y la Internet, y dirige el tráfico de datos entre ellas.

Modem

Un módem (modulador-demodulador) es un dispositivo que convierte las señales digitales de una computadora en señales analógicas para su transmisión a través de líneas telefónicas, de cable o de fibra óptica (modulación), y viceversa para recibir y decodificar las señales (demodulación).

Protocolo IP

El Protocolo de Internet (IP) es un conjunto de reglas que define cómo los datos se envían y se reciben a través de Internet y otras redes, dividiendo la información en paquetes que son enrutados y direccionados a su destino utilizando una dirección IP única para cada dispositivo.

HTTP

HTTP, o Protocolo de Transferencia de Hipertexto, es un protocolo de capa de aplicación que permite la comunicación entre navegadores web (clientes) y servidores para transferir recursos como páginas web en la World Wide Web.

Endpoint

Un endpoint es un punto de conexión o un dispositivo que puede enviar o recibir datos dentro de una red, sirviendo como una entrada o salida de información.

Realidad virtual

La Realidad Virtual (RV) es una tecnología que crea un entorno tridimensional simulado y generado por computadora, permitiendo a los usuarios sumergirse en él e interactuar usando dispositivos especializados como cascos VR y sensores.

Realidad aumentada

La tecnología de Realidad Aumentada (RA) es un sistema que fusiona el mundo real con información digital generada por computadora, superponiendo elementos virtuales (como gráficos, sonidos o texto) sobre el entorno físico de un usuario en tiempo real.

Pitch elevator

Un "Pitch Elevator" o "Discurso del Ascensor" para tecnología es una presentación concisa y persuasiva de una idea o producto tecnológico, diseñada para captar la atención de inversores, clientes o colaboradores en un corto período de tiempo, como el que dura un viaje en ascensor (aproximadamente 30 segundos a 2 minutos).

Mantenimiento predictivo

El mantenimiento predictivo es una tecnología y estrategia que utiliza sensores, Internet de las Cosas (IoT), inteligencia artificial (IA) y análisis de datos en tiempo real para monitorear equipos y predecir fallas potenciales antes de que ocurran.

Mantenimiento correctivo

El mantenimiento correctivo en tecnología es una acción reactiva que se realiza para solucionar fallos y averías en equipos y sistemas informáticos una vez que se han producido, con el objetivo de restaurar su funcionamiento normal lo antes posible.

Renderizar

La renderización en tecnología es el proceso de generar una representación visual o auditiva (como imágenes, videos o animaciones) a partir de datos o modelos tridimensionales.

Overclock

El overclocking es una técnica tecnológica para aumentar el rendimiento de un componente de hardware, como la CPU o la GPU, haciendo que opere a una velocidad de reloj superior a la especificada por el fabricante.

Vram

La VRAM (Memoria de Acceso Aleatorio de Video) es un tipo de memoria especializada y de alta velocidad ubicada en la tarjeta gráfica de una computadora, cuya función es almacenar temporalmente los datos gráficos, como imágenes, texturas y sombras, para que la GPU (Unidad de Procesamiento Gráfico) pueda acceder a ellos rápidamente y procesarlos para la visualización en la pantalla.

Micropuntos

La tecnología de Micropuntos es un método de identificación y seguimiento de objetos mediante la aplicación de diminutas etiquetas o marcas (micropuntos) que contienen información única, como números de serie o identificadores personalizados, sobre piezas o activos.

Computadores cuánticos

La computación cuántica es un tipo de computación que utiliza los principios de la mecánica cuántica, como la superposición y el entrelazamiento, para realizar cálculos.

Networking

En el ámbito de la tecnología, el networking se refiere al proceso de conectar múltiples dispositivos y sistemas, como computadoras, servidores y otros equipos, para que puedan comunicarse, intercambiar datos y compartir recursos.

Huella digital

La huella digital en tecnología es el rastro de datos que una persona o negocio deja al usar Internet, abarcando tanto la información que se comparte voluntariamente (huella activa) como los datos recopilados de manera pasiva sin consentimiento explícito (huella pasiva).

Codigo ASCII

El código ASCII (American Standard Code for Information Interchange) es un sistema de codificación de caracteres que asigna un valor numérico a letras, números, signos de puntuación y otros símbolos, permitiendo a las computadoras y dispositivos comunicarse e intercambiar información de forma estándar.

UHS

La tecnología UHS (Ultra High Speed) se refiere a una interfaz de comunicación para tarjetas de memoria SD y microSD, que ofrece velocidades de transferencia de datos considerablemente más rápidas que las clases estándar, permitiendo grabar y transferir contenido de alta resolución como video 4K de manera fluida y eficiente.

Sistema operativo

Un sistema operativo es el software fundamental de una computadora o dispositivo móvil que actúa como intermediario entre el usuario y el hardware, gestionando sus recursos (CPU, memoria, almacenamiento, periféricos) y permitiendo la ejecución de aplicaciones.

Sistema operativo rudimentario

Un sistema operativo rudimentario se refiere a un sistema operativo que es básico, simple o con funcionalidades limitadas, diseñado para un hardware con recursos restringidos o para una tarea muy específica.

GIT

Git es un proyecto de código abierto de calidad. Los encargados de mantener el proyecto han demostrado un criterio equilibrado y un enfoque maduro para satisfacer las necesidades a largo plazo de sus usuarios con publicaciones periódicas que mejoran la facilidad de uso y la funcionalidad.

Plagio informático

El plagio informático es el acto de tomar y presentar como propio código fuente, algoritmos, software u otros contenidos digitales sin reconocer o citar al autor original, aprovechando las tecnologías de la información y la vasta cantidad de contenido disponible en Internet.

Phubbing

El phubbing es el acto de ignorar a una persona que está presente al priorizar el uso de un teléfono móvil o cualquier otro dispositivo electrónico.

"Axiedad" es un término, no una palabra reconocida oficialmente, que describe un estado de tensión y confusión causado por la creencia de que se están recibiendo mensajes o notificaciones en el celular, aunque esto no esté ocurriendo realmente.

El vamping es una práctica que consiste en el uso excesivo de dispositivos electrónicos durante la noche, robando horas de sueño necesarias para el descanso y el bienestar.

Delitos informaticos

Los delitos informáticos son conductas ilegales y antijurídicas realizadas a través de medios digitales, como computadoras y redes, que buscan causar daños, obtener beneficios ilícitos, o violar la confidencialidad, integridad y disponibilidad de la información y los sistemas informáticos.

La ciberdependencia es una adicción a las tecnologías de la información y la comunicación (TIC), caracterizada por un uso excesivo y compulsivo de dispositivos y redes sociales que afecta negativamente la vida personal, social y laboral del individuo.

Base de datos

En tecnología, una base de datos es una colección organizada y sistemática de información digital, que puede incluir palabras, números, imágenes, videos y otros tipos de datos.

GPU

Una GPU (Unidad de Procesamiento Gráfico) es un circuito electrónico especializado para realizar cálculos matemáticos a alta velocidad, inicialmente para la renderización de imágenes y vídeos, pero ahora también para tareas de alto cómputo como la inteligencia artificial.

CPU

En tecnología, la CPU (Unidad Central de Procesamiento) es el "cerebro" de un sistema informático, responsable de ejecutar instrucciones y realizar cálculos para que funcionen los programas y el sistema operativo.

RAM

En tecnología, RAM significa Memoria de Acceso Aleatorio (Random Access Memory). Es un tipo de memoria volátil y temporal que el ordenador usa para almacenar datos e instrucciones que está procesando activamente.

SSD

Una SSD (Unidad de Estado Sólido, por sus siglas en inglés Solid State Drive) es un dispositivo de almacenamiento que usa memoria flash para guardar datos, careciendo de partes mecánicas móviles como los discos duros (HDD), lo que le permite ser más rápido, silencioso, duradero y consumir menos energía.

HDD

En tecnología, HDD son las siglas en inglés de Hard Disk Drive (Unidad de Disco Duro), un dispositivo de almacenamiento de datos no volátil que utiliza discos giratorios con magnetismo para guardar de forma persistente el sistema operativo, aplicaciones y archivos.

USB

En tecnología, USB (Universal Serial Bus) es un estándar de interfaz que permite conectar dispositivos electrónicos a computadoras y otros sistemas para la transferencia de datos y suministro de energía.

Teclado táctil

En tecnología, un teclado táctil puede referirse a dos conceptos: la mecanografía táctil (o touch typing), que es la habilidad de escribir en un teclado físico sin mirar las teclas para mayor rapidez y precisión, o a un teclado en pantalla (o teclado virtual), una interfaz visual en un dispositivo con pantalla táctil que permite al usuario ingresar datos al tocar los caracteres en la pantalla con un dedo o un lápiz óptico.

Mouse

En tecnología, un mouse es un dispositivo apuntador de entrada que permite al usuario controlar un cursor en la pantalla de un ordenador o portátil, interactuar con la interfaz gráfica (GUI) mediante acciones como hacer clic y arrastrar, y navegar por documentos y páginas web usando sus botones y rueda de desplazamiento.

Teclado mecanico

Un teclado mecánico es un tipo de teclado que utiliza interruptores mecánicos individuales para cada tecla, en lugar de una membrana de goma o silicona, como los teclados convencionales.

Teclado de membrana

Un teclado de membrana es un tipo de teclado que utiliza una o varias láminas flexibles con pistas conductoras para registrar las pulsaciones de teclas, en lugar de interruptores mecánicos individuales.

Hz

En tecnología, Hz (hercio) es la unidad de medida de la frecuencia. Se refiere al número de veces que ocurre un evento cíclico por segundo, y se aplica comúnmente a la frecuencia de actualización de monitores (cuántas imágenes pueden mostrar por segundo), la velocidad de reloj de procesadores (cuántos ciclos puede completar por segundo) y la frecuencia de señales como ondas de radio o de red eléctrica.

Celulares

En tecnología, un celular es un dispositivo electrónico portátil que permite la comunicación inalámbrica a través de ondas de radio, facilitando llamadas de voz, mensajes de texto y acceso a internet.

Audífonos

En tecnología, los audífonos (también llamados audiófonos o sonotones) son dispositivos electrónicos diseñados para amplificar y modificar señales sonoras, ayudando a personas con pérdida auditiva a escuchar mejor, comunicarse con mayor facilidad y participar plenamente en sus actividades diarias.

Parlantes

En tecnología, los parlantes (también conocidos como altavoces, bocinas o altoparlantes) son dispositivos que convierten señales eléctricas en ondas sonoras, permitiendo escuchar el audio de diferentes fuentes como computadoras, teléfonos y televisores.

Cargador

En tecnología, la palabra "cargador" tiene dos significados principales: por un lado, es un dispositivo físico que suministra energía a otro dispositivo (como un smartphone o portátil) para recargar sus baterías, ya sea a través de un cable o de forma inalámbrica.

Fuente de poder

En tecnología, una fuente de poder (o fuente de alimentación) es un componente electrónico que convierte la corriente alterna (CA) de la red eléctrica a corriente continua (CC) de los voltajes correctos para que funcionen los dispositivos electrónicos, como computadoras, televisores e impresoras.

Motherboard

Una motherboard es la tarjeta de circuito impreso principal de una computadora, actuando como la columna vertebral que conecta y permite la comunicación entre todos los componentes esenciales, como el procesador (CPU), la memoria RAM, el almacenamiento y los puertos para dispositivos externos.

Bluetooth

Bluetooth es una tecnología de comunicación inalámbrica de corto alcance que permite a dos o más dispositivos intercambiar datos y sincronizar información sin necesidad de cables, basándose en ondas de radio de baja potencia.

Webcam

Una webcam es una cámara digital que captura vídeo y audio y los transmite en tiempo real a través de Internet, conectándose usualmente a una computadora o portátil mediante puertos USB o de forma inalámbrica.

Scanner

En tecnología, un escáner (o scanner) es un dispositivo o periférico de entrada que digitaliza contenido físico, como documentos, imágenes o hasta objetos, convirtiéndolo en datos digitales para su visualización, edición y almacenamiento en una computadora o para su procesamiento en otros sistemas.

Pantallas tactiles

Una pantalla táctil es un tipo de pantalla electrónica que permite al usuario interactuar directamente con un dispositivo tocando, deslizando o arrastrando el dedo o un lápiz óptico sobre su superficie, eliminando la necesidad de periféricos como teclados y ratones.

Monitor

En tecnología, un monitor es un dispositivo de salida que funciona como una pantalla, mostrando de forma visual la información procesada por una computadora, permitiendo al usuario interactuar con el sistema.

Impresora

En informática, una impresora es un dispositivo periférico de salida que se conecta a una computadora para convertir información digital, como texto y gráficos, en una representación física permanente, generalmente sobre papel, mediante tecnologías como la inyección de tinta, láser, o impresión 3D.

Proyector

En tecnología, un proyector es un dispositivo electrónico que recibe una señal de video de una fuente (como una computadora o un reproductor multimedia) y la proyecta sobre una superficie grande, como una pantalla o pared, ampliando la imagen o el video para ser visto por una audiencia.

CD

En tecnología, CD es la abreviatura de Disco Compacto (Compact Disc), un medio de almacenamiento óptico de datos digitales, originalmente diseñado para audio, pero que evolucionó para guardar música, video, software y otros tipos de información en formatos como CD-ROM (solo lectura), CD-R (grabable) y CD-RW (regrabable).

DVD

Un DVD (disco versátil digital o disco de video digital) es un disco óptico de 12 cm de diámetro introducido en los años 90, capaz de almacenar grandes cantidades de datos digitales, como películas, música, software y documentos.

Blu-ray

En tecnología, Blu-ray es un disco óptico de alta capacidad diseñado para almacenar y reproducir contenido de alta definición y otros datos digitales, siendo el sucesor del DVD.

Núcleos

En tecnología, el término "núcleo" puede referirse a dos conceptos principales: un núcleo de CPU (Unidad Central de Procesamiento), que es la unidad de procesamiento física e independiente dentro de un chip que ejecuta instrucciones y procesa datos simultáneamente, y un núcleo de sistema operativo (kernel), que es el componente central de un sistema operativo responsable de gestionar el hardware, la memoria y las tareas de la computadora.

Procesadores lógicos

Un procesador lógico es una unidad del sistema operativo para ejecutar instrucciones y coordinar tareas en un procesador físico, que puede ser un núcleo real o un núcleo virtual creado mediante tecnologías como el Hyper-Threading.

Chipset

Un chipset en tecnología es un conjunto de circuitos integrados en la placa base de un ordenador que actúa como un "controlador de tráfico" o "pegamento" para la placa.

burn in

TV

En tecnología, la TV (Televisión) es un sistema de transmisión y recepción de imágenes en movimiento y sonido a distancia que utiliza señales digitales o analógicas para difundir contenidos a través de diferentes medios como la televisión abierta, satélite, cable o Internet.

Mouse mecánico

Videojuegos

Steam

Epic games

Logitech

Logitech es una empresa suiza multinacional líder en el diseño y fabricación de periféricos y accesorios tecnológicos para computadoras, dispositivos móviles y otros equipos electrónicos.

Razer

En el ámbito de la tecnología, Razer es una marca reconocida a nivel mundial por diseñar, desarrollar y vender hardware y software de consumo de alta gama, enfocado principalmente en la comunidad gamer.

Ergonómico

"ergonómico" describe el diseño de dispositivos, software y espacios de trabajo para adaptarse a las necesidades y capacidades humanas, con el fin de optimizar la comodidad, la eficiencia y la seguridad del usuario, reduciendo el riesgo de lesiones y mejorando el bienestar.

Youtube

En tecnología, YouTube es una plataforma web de video-streaming y una red social de contenido audiovisual, propiedad de Google, que permite a los usuarios subir, ver, compartir y comentar videos de forma gratuita.

Tiktok

En el ámbito de la tecnología, TikTok es una plataforma de redes sociales de origen chino que permite a los usuarios crear, compartir y descubrir vídeos cortos en formato vertical, con herramientas integradas para la edición, música y efectos especiales.

Twitter

En tecnología, Twitter (ahora llamado X) es una red social de microblogging que permite a los usuarios compartir mensajes cortos llamados "tweets" o publicaciones, de un máximo de 280 caracteres, que pueden incluir texto, fotos y videos.

WeChat

Weibo

Douyin

Decodificación

En tecnología, decodificación es el proceso de convertir información que fue codificada, comprimida o cifrada a su formato original o comprensible.

Almacenamiento

En tecnología, el almacenamiento se refiere a la conservación de datos digitales en medios o sistemas específicos para que puedan ser accesibles y utilizados cuando se necesiten.

Memoria caché

La memoria caché es un almacenamiento temporal de alta velocidad que guarda datos de acceso frecuente para que el procesador pueda acceder a ellos de forma más rápida que desde el disco duro o la memoria RAM principal.

ALU

Una Unidad Aritmético-Lógica (ALU, por sus siglas en inglés) es un componente fundamental de la unidad central de procesamiento (CPU) de una computadora, encargado de realizar operaciones matemáticas (aritméticas) y de comparación y toma de decisiones (lógicas) con datos binarios.

Dual-core

En tecnología, "Dual-core" (doble núcleo) se refiere a un procesador que tiene dos unidades de procesamiento completas e independientes (núcleos) dentro de un solo chip.

Multitarea

La multitarea en tecnología es la capacidad de un sistema o usuario para manejar y ejecutar múltiples tareas o procesos de forma concurrente o simultánea, permitiendo la alternancia y gestión eficiente de diferentes aplicaciones o actividades para mejorar la productividad.

Smartphone

Un smartphone o teléfono inteligente es un dispositivo móvil que combina las funciones de un teléfono celular tradicional con las de una computadora de bolsillo, gracias a su sistema operativo avanzado que permite instalar y ejecutar aplicaciones, acceder a internet, gestionar correo electrónico y otras tareas de computación, además de sus capacidades multimedia como cámara, reproducción de video y música.

Setup

En tecnología, "Setup" se refiere al proceso de configuración, instalación y ajuste de hardware y software, o a un programa específico que facilita esta tarea.

Pantalla plana

Una pantalla plana es una tecnología de visualización electrónica ligera y delgada, de menos de 10 cm de grosor, que ofrece imágenes y texto con alta resolución y contraste, reemplazando a las pantallas cóncavas tradicionales de tubos de rayos catódicos (CRT).

Arpanet

ARPANET (Advanced Research Projects Agency Network) fue una red informática pionera creada por el Departamento de Defensa de EE. UU. en la década de 1960 que sirvió como precursora de Internet.

Internet ubicuo

El Internet ubicuo se refiere a una red de comunicación que, gracias a la computación ubicua, está disponible de forma constante y generalizada en cualquier momento y lugar, integrando la tecnología informática en los objetos y dispositivos cotidianos para ofrecer una experiencia conectada y siempre activa.

Servidor

En informática, un servidor es una computadora o programa diseñado para proporcionar recursos, servicios y datos a otras computadoras o dispositivos, llamados clientes, a través de una red.

Tarjeta de red

Una tarjeta de red es un componente de hardware que permite a un ordenador o dispositivo conectarse y comunicarse con otras máquinas en una red local (LAN) o en Internet, tanto de forma cableada como inalámbrica.

Unidad óptica

Una unidad óptica es un dispositivo de hardware informático que utiliza un láser para leer y escribir información en discos ópticos como CD, DVD y Blu-ray, permitiendo el acceso a música, películas, software y otros datos.

Wi-Fi

Wi-Fi (del inglés Wireless Fidelity, o fidelidad inalámbrica) es una tecnología de red local inalámbrica que permite a dispositivos electrónicos como computadoras y teléfonos inteligentes conectarse a Internet o entre sí sin necesidad de cables

Aplicación

En tecnología, una aplicación (o app) es un programa informático diseñado para que los usuarios realicen tareas o funciones específicas, a diferencia de los sistemas operativos, que gestionan el funcionamiento del dispositivo.

Actualización

En tecnología, una actualización es la liberación de una nueva versión de software o sistema que reemplaza a una anterior, introduciendo mejoras, nuevas funciones, corrección de errores o parches de seguridad para optimizar el rendimiento y la funcionalidad de un dispositivo o aplicación.

Open source

En tecnología, Open Source (código abierto) se refiere a un modelo de desarrollo de software donde el código fuente es accesible, modificable y distribuible libremente por cualquier persona bajo una licencia>> apropiada.

Backup

En tecnología, un backup o copia de seguridad es el proceso de crear copias de datos de un sistema, que se almacenan por separado de los originales con el fin de poder restaurar la información en caso de pérdidas o daños accidentales, como fallas del sistema, errores humanos, ciberataques o desastres naturales.

Restore

En tecnología, "Restore" (o restaurar) es el proceso de recuperar datos, sistemas o configuraciones desde una copia de seguridad (backup) para devolverlos a un estado anterior o funcional, normalmente después de una pérdida de datos, un fallo del sistema o para revertir cambios no deseados.

Instalación

En tecnología, la instalación es el proceso de preparar e implementar un sistema, hardware o software para su correcto funcionamiento, lo que incluye la conexión física, la configuración de parámetros y la transferencia de archivos necesarios para que opere.

Desinstalación

En tecnología, la desinstalación es el proceso de eliminar un programa o aplicación de software de un sistema informático o dispositivo, borrando todos los archivos y componentes asociados con él para liberar espacio y mejorar el rendimiento del sistema.

Parche

Un parche o solución de software es un trabajo de reparación rápida para una pieza de programación, diseñada para resolver problemas de funcionalidad, mejorar la seguridad o agregar nuevas funciones.

DNS

En tecnología, el DNS (Sistema de Nombres de Dominio) es el "directorio telefónico" de Internet, ya que traduce los nombres de dominio fáciles de recordar (como www.google.com) en las direcciones IP numéricas que las computadoras usan para localizar y conectarse a un sitio web.

HTTPS

HTTPS es el protocolo de transferencia de hipertexto seguro que cifra la comunicación entre tu navegador y un sitio web, garantizando la privacidad y autenticidad de los datos.

FTP

En tecnología, FTP (Protocolo de Transferencia de Archivos) es un protocolo de red estándar que permite transferir archivos entre dos computadoras, un cliente y un servidor, a través de una red basada en TCP, como Internet.

LAN

Una LAN (Local Area Network o red de área local) es una red de ordenadores y dispositivos interconectados dentro de un área geográfica limitada, como un hogar, oficina o edificio, que permite compartir recursos y datos a alta velocidad.

WAN

Una WAN (Red de Área Extensa) es un tipo de red informática que conecta múltiples redes de área local (LAN) a través de grandes distancias geográficas, como ciudades, países o incluso el mundo entero.

Proxy

En tecnología, un servidor proxy es un intermediario entre un usuario e Internet que reenvía solicitudes de datos y responde a ellas, actuando como un puente para ofrecer beneficios como el enmascaramiento de la dirección IP, el filtrado de contenido, la mejora de la seguridad y la optimización del rendimiento mediante el almacenamiento en caché.

NAT

En tecnología, NAT (Network Address Translation o Traducción de Direcciones de Red) es un proceso que permite a múltiples dispositivos de una red privada compartir una única dirección IP pública para acceder a Internet.

Ping

En tecnología, el Ping es tanto el tiempo que tarda un paquete de datos en ir y volver entre dos dispositivos en una red (medido en milisegundos), como la herramienta que se utiliza para enviar esta solicitud y medir esa latencia.

Packet

En tecnología, un paquete es un bloque de datos estructurado que se utiliza para transmitir información a través de una red informática, dividiendo un mensaje más grande en segmentos más pequeños y manejables.

Latencia

En tecnología, la latencia es el retraso o el tiempo que tarda un paquete de datos en viajar desde su punto de origen hasta su destino, o en general, entre un estímulo y la respuesta producida.

Ancho de banda

En tecnología, ancho de banda se refiere a la capacidad máxima de datos que puede transmitirse a través de una conexión o sistema en un período de tiempo determinado, como una conexión a internet.

Red inalámbrica

Una red inalámbrica es un sistema que permite conectar dispositivos informáticos y periféricos a una red sin necesidad de cables, utilizando ondas de radiofrecuencia para transmitir datos.

Red cableada

Una red cableada es un sistema informático donde los dispositivos se conectan a través de cables físicos para transferir datos y compartir recursos, como cables Ethernet y fibra óptica.

Cloud computing

El cloud computing o computación en la nube es la entrega de recursos informáticos —como servidores, almacenamiento, bases de datos, software, análisis e inteligencia— a través de Internet ("la nube"), bajo demanda y mediante un modelo de pago por uso.

Phishing

El phishing es un tipo de ciberataque donde los delincuentes se hacen pasar por entidades o personas de confianza para engañar a las víctimas y obtener información confidencial, como contraseñas, datos bancarios o números de tarjetas de crédito.

Ransomware

El ransomware es un tipo de malware que retiene como rehenes los datos confidenciales o el dispositivo de una víctima, amenazando con mantenerlos bloqueados, o algo peor, a menos que la víctima pague un rescate al atacante.

Spyware

El spyware es un tipo de software malicioso que se instala en un dispositivo sin el conocimiento ni consentimiento del usuario para recopilar información personal y enviarla a terceros.

Troyano

En tecnología, un troyano (o caballo de Troya) es un tipo de malware que se disfraza de programa o archivo legítimo para engañar al usuario y obtener acceso no autorizado a su sistema informático.

Autenticación

En tecnología, la autenticación es el proceso de verificar la identidad de un usuario, dispositivo o servicio, para asegurar que solo aquellos autorizados puedan acceder a recursos protegidos.

Autorización

En tecnología, la autorización es el proceso de determinar qué derechos y permisos tiene un usuario o sistema para acceder a un recurso o ejecutar una acción específica dentro de un sistema informático, y se realiza después de la autenticación.

Cifrado

En tecnología, el cifrado es un proceso que codifica la información haciéndola ilegible para cualquier persona no autorizada, utilizando un algoritmo matemático y una clave secreta.

SSL

En tecnología, SSL (Secure Sockets Layer) es un protocolo de seguridad que crea un canal de comunicación cifrado entre un servidor y un cliente (como un navegador web), garantizando la privacidad, autenticación e integridad de los datos transmitidos en Internet.

TLS

TLS, o Seguridad de la Capa de Transporte, es un protocolo de seguridad que cifra y autentica las comunicaciones en Internet para proteger los datos de accesos no autorizados y asegurar la integridad de la información.

Biometría

En tecnología, la biometría es la medición y el análisis de características biológicas o conductuales únicas de una persona para autenticar su identidad.

Token

En tecnología, un token es una representación digital de valor o un identificador seguro, que puede ser un dispositivo físico, una clave digital o una unidad de datos.

Contraseña

Una contraseña, clave o código de acceso​ es una forma de autentificación que utiliza información secreta para controlar el acceso hacia algún recurso.

Doble factor

El doble factor en tecnología, o autenticación de dos factores (2FA), es una medida de seguridad que requiere que un usuario proporcione dos formas diferentes de verificación para acceder a una cuenta o sistema, más allá de la contraseña inicial.

Seguridad perimetral

La seguridad perimetral en tecnología se refiere al conjunto de medidas, herramientas y sistemas que se implementan para proteger los límites de una red informática o una infraestructura física de accesos no autorizados y amenazas externas e internas.

NoSQL

NoSQL (No Solo SQL) se refiere a una categoría de bases de datos no relacionales que almacenan datos en formatos no tabulares, como documentos, clave-valor, columnas amplias o gráficos, en lugar de la estructura rígida de tablas y filas de las bases de datos relacionales tradicionales.

Big data

En tecnología, el Big Data se define como la gestión y análisis de enormes volúmenes de datos, de alta velocidad y variedad (estos son los "tres V": Volumen, Velocidad y Variedad), que superan las capacidades de las herramientas tradicionales de procesamiento.

Data mining

El Data Mining o minería de datos en tecnología es el proceso de extraer información valiosa, patrones y conocimientos ocultos de grandes volúmenes de datos, utilizando técnicas de inteligencia artificial, machine learning y estadística.

Data warehouse

Un Data Warehouse (o Almacén de Datos) es un sistema centralizado de almacenamiento que integra, organiza y procesa grandes volúmenes de datos históricos y actuales de diversas fuentes operativas de una organización.

ETL

En tecnología, ETL significa Extract, Transform, Load (Extraer, Transformar, Cargar) y es un proceso de integración de datos que recopila información de diversas fuentes, la limpia y la organiza según reglas específicas y luego la carga en un repositorio centralizado, como un almacén de datos (data warehouse), listo para ser analizado.

Sharding

En tecnología, el Sharding (fragmentación) es una técnica de escalado que divide un conjunto de datos grande en subconjuntos más pequeños y manejables llamados "shards" o fragmentos, los cuales se distribuyen entre varios servidores o nodos de bases de datos.

Correo electrónico

En tecnología, el correo electrónico (o e-mail) es un servicio de red que permite a los usuarios enviar y recibir mensajes digitales a través de redes informáticas como Internet, utilizando un dispositivo electrónico y una cuenta de correo con una dirección única.

Criptografía

En tecnología, la criptografía es la práctica de codificar información utilizando algoritmos para asegurar que solo las personas autorizadas puedan acceder a ella, garantizando así la confidencialidad, integridad y autenticidad de los datos, tanto en reposo como en tránsito o uso.

Ethernet

Ethernet es una tecnología de red que utiliza cables (como los cables RJ45) para conectar dispositivos electrónicos como computadoras, impresoras y enrutadores a una red de área local (LAN) o a una red de área amplia (WAN).

Criptomonedas

Las criptomonedas son activos digitales descentralizados, ya que no están controladas ni respaldadas por ningún banco central, y sus intercambios no requieren de intermediarios.

Metaverso

El metaverso es un universo digital persistente e interconectado donde las personas pueden interactuar entre sí y con entornos virtuales de forma inmersiva a través de avatares.

5G

El 5G es la quinta generación de tecnología de redes móviles inalámbricas, un estándar global que superó al 4G y está diseñado para conectar prácticamente a todo, desde personas y dispositivos hasta máquinas y objetos.

C++

En tecnología, C++ es un lenguaje de programación compilado de propósito general, multiparadigma y de alto rendimiento que se utiliza para crear software eficiente y robusto.

Intérprete

En tecnología, un intérprete es un programa que ejecuta un código fuente línea por línea, actuando como una interfaz directa entre el código y el procesador, sin necesidad de generar un archivo ejecutable independiente.

Máquina virtual

Una máquina virtual (VM) es un ordenador virtualizado y basado en software que se ejecuta sobre hardware físico, pero que actúa como un equipo independiente con su propio sistema operativo y aplicaciones.

Cliente virtual

En tecnología, un cliente virtual puede referirse a dos conceptos principales: un cliente digital, que es una persona que interactúa y compra productos o servicios en línea a través de diversos medios digitales; o un cliente virtualizado, que es un software o sistema operativo alojado en una máquina virtual dentro del equipo del usuario, permitiendo ejecutar diferentes entornos de forma simultánea.

Hosting

En tecnología, el hosting (o alojamiento web) es un servicio que permite a usuarios y organizaciones publicar sitios web en Internet, almacenando sus archivos (texto, imágenes, videos, bases de datos) en un servidor conectado a la red para que estén accesibles desde cualquier parte del mundo las 24 horas del día.

Dominio

En informática, un dominio es la dirección única y fácil de recordar de un sitio web en internet, que se utiliza en lugar de una compleja dirección IP.

URL

En tecnología, una URL (Localizador Uniforme de Recursos) es la dirección web que especifica la ubicación de un recurso único en Internet, como una página web, un documento, una imagen o un archivo, y el protocolo utilizado para acceder a él.

Transfer Learning avanzado

Técnica que reutiliza el conocimiento adquirido en un modelo para aplicarlo a nuevas tareas con menos datos.

Multi-Task Learning

Entrenamiento de un modelo en múltiples tareas relacionadas para mejorar la generalización.

Continual Learning

Capacidad de un modelo para aprender de manera incremental sin olvidar conocimiento previo.

Catastrophic Forgetting

Problema en aprendizaje continuo donde un modelo pierde desempeño en tareas anteriores al aprender nuevas.

Federated Averaging

Método de agregación de parámetros en aprendizaje federado, que combina modelos entrenados en múltiples dispositivos.

Gradient Compression

Técnicas para reducir el tamaño de gradientes en entrenamiento distribuido y acelerar comunicación.

Knowledge Distillation

Proceso de transferir conocimiento de un modelo grande a uno más pequeño y eficiente.

Model Pruning

Reducción del tamaño de un modelo eliminando parámetros redundantes sin perder demasiada precisión.

Quantization (IA)

Técnica que reduce la precisión numérica de los parámetros de un modelo para optimizar rendimiento.

Low-Rank Adaptation (LoRA)

Método eficiente de ajuste de modelos grandes mediante capas de bajo rango que reducen costos de cómputo.

Sparse Models

Modelos de IA que utilizan matrices dispersas para reducir memoria y aumentar eficiencia.

Mixture of Experts (MoE)

Arquitectura en la que múltiples submodelos especializados se activan selectivamente para tareas específicas.

Parameter-Efficient Fine-Tuning (PEFT)

Estrategia para adaptar grandes modelos entrenando solo una fracción de los parámetros.

Prompt Tuning

Optimización de vectores de entrada aprendidos para mejorar el desempeño en modelos de lenguaje.

In-Context Learning

Capacidad de los LLMs para aprender patrones a partir de ejemplos provistos en la entrada sin ajuste de parámetros.

Chain-of-Thought Prompting

Técnica que guía a los modelos de lenguaje a razonar paso a paso en lugar de dar solo la respuesta final.

Self-Consistency Decoding

Método de inferencia en modelos de lenguaje que explora múltiples trayectorias de razonamiento antes de elegir la mejor.

Temperature en Sampling

Parámetro en generación de texto que controla la aleatoriedad: valores bajos producen respuestas más deterministas.

Top-k Sampling

Método de generación de texto que selecciona la próxima palabra entre las k más probables.

Top-p (Nucleus) Sampling

Selección dinámica de palabras en generación de texto basada en la probabilidad acumulada.

Beam Search

Algoritmo de búsqueda que explora múltiples secuencias candidatas en generación de texto para elegir la más probable.

Greedy Decoding

Estrategia de generación de texto que elige siempre la palabra más probable en cada paso.

Contrastive Search

Método de inferencia que busca equilibrio entre diversidad y coherencia en texto generado.

Reinforcement Learning (avanzado)

Área de IA donde los agentes aprenden a tomar decisiones óptimas mediante recompensas y penalizaciones.

Policy Gradient Methods

Algoritmos de RL que optimizan directamente la política de un agente en lugar del valor esperado.

Actor-Critic Methods

Algoritmos de RL que combinan estimadores de valor (crítico) y políticas (actor) para mejorar estabilidad.

Deep Q-Networks (DQN)

Algoritmos que combinan Q-learning con redes neuronales profundas para problemas complejos.

Multi-Agent Reinforcement Learning

Extensión del RL donde múltiples agentes interactúan y aprenden en entornos compartidos.

Imitation Learning

Entrenamiento de agentes a partir de la observación y reproducción del comportamiento humano.

Inverse Reinforcement Learning

Técnica que infiere la función de recompensa observando el comportamiento de expertos.

Hierarchical Reinforcement Learning

Descomposición de tareas complejas en sub-tareas más manejables dentro de un marco de RL.

Exploration vs Exploitation

Dilema en RL entre probar nuevas estrategias (exploración) y optimizar las ya conocidas (explotación).

Reward Shaping

Diseño de señales de recompensa adicionales para guiar el aprendizaje de agentes de RL.

Curriculum Learning

Técnica que entrena modelos comenzando con tareas simples y aumentando progresivamente la dificultad.

Self-Supervised Learning

Método de aprendizaje automático que genera etiquetas automáticamente a partir de los datos mismos.

Contrastive Learning

Enfoque de autoaprendizaje que entrena modelos a distinguir similitudes y diferencias entre pares de datos.

SimCLR / MoCo

Métodos contrastivos para aprendizaje de representaciones visuales sin supervisión explícita.

BYOL / SimSiam

Técnicas de autoaprendizaje que eliminan la necesidad de comparaciones negativas en aprendizaje contrastivo.

Masked Autoencoders

Modelos de autoaprendizaje que reconstruyen partes faltantes de datos de entrada, como en BERT o MAE para imágenes.

Vision Transformers (ViT)

Arquitectura que aplica mecanismos de atención de Transformers en visión por computadora, reemplazando CNNs en varias tareas.

Swin Transformer

Modelo de visión basado en Transformers que utiliza ventanas deslizantes para reducir el costo computacional.

DETR (Detection Transformer)

Arquitectura que aplica Transformers a la detección de objetos, eliminando la necesidad de técnicas clásicas como anchors.

CLIP (Contrastive Language-Image Pretraining)

Modelo multimodal entrenado para asociar texto con imágenes mediante aprendizaje contrastivo.

DALL·E

Modelo generativo capaz de crear imágenes a partir de descripciones textuales.

Stable Diffusion

Modelo de difusión que genera imágenes de alta calidad a partir de texto mediante denoising progresivo.

Imagen (Google)

Modelo de difusión enfocado en generación fotorrealista de imágenes desde texto.

MidJourney

Plataforma de IA generativa especializada en creación artística y diseño visual.

Generative Adversarial Networks (GANs)

Modelos que enfrentan una red generadora contra una discriminadora para producir datos realistas.

Conditional GANs

Extensión de GANs que genera datos condicionados en etiquetas o atributos específicos.

StyleGAN

Familia de GANs que permite controlar estilos y detalles en generación de imágenes.

CycleGAN

Modelo GAN que realiza traducción de imágenes entre dos dominios sin necesidad de pares alineados.

Pix2Pix

Modelo de traducción de imágenes entrenado en pares, como convertir bocetos en fotos.

Diffusion Models

Modelos generativos que transforman ruido en datos estructurados mediante un proceso de denoising iterativo.

Score-Based Models

Modelos de difusión que aprenden distribuciones de probabilidad a partir de gradientes de densidad.

Variational Autoencoders (VAE)

Modelos probabilísticos que aprenden distribuciones latentes de datos para generación y compresión.

Normalizing Flows

Modelos generativos que aprenden transformaciones invertibles entre distribuciones simples y complejas.

Energy-Based Models

Modelos que asignan un valor de energía a configuraciones de datos, aprendiendo distribuciones implícitas.

Deep Fake

Tecnología que utiliza IA para generar falsificaciones realistas de rostros, voces o videos.

Voice Cloning

Técnica de síntesis de voz que replica el timbre y estilo de una persona con pocos datos.

Text-to-Speech (TTS)

Sistemas que convierten texto escrito en voz natural.

Speech-to-Text (STT)

Tecnología de reconocimiento de voz que transcribe audio a texto escrito.

Neural Machine Translation (NMT)

Sistemas de traducción automática basados en redes neuronales profundas.

Multimodal AI

Inteligencia artificial que integra múltiples tipos de datos: texto, imágenes, audio y video.

Audio Transformers

Modelos basados en Transformers diseñados para tareas de audio como clasificación y síntesis.

Whisper (OpenAI)

Modelo de reconocimiento de voz multilingüe robusto y entrenado en gran escala.

HuBERT

Wav2Vec 2.0

BCI (Brain-Computer Interface)

fNIRS (Functional Near-Infrared Spectroscopy)

Técnica no invasiva para medir actividad cerebral, usada en combinación con IA.

Proceso de interpretar señales cerebrales con IA para inferir intenciones o pensamientos.

Tecnología experimental que convierte directamente señales cerebrales en texto.

Brain-to-Speech

Investigación para generar voz a partir de actividad cerebral decodificada.

Neuroprosthetics

Dispositivos controlados por IA y señales cerebrales que sustituyen funciones motoras perdidas.

Cognitive Computing

IA inspirada en procesos cognitivos humanos para mejorar razonamiento y toma de decisiones.

Explainable AI (XAI)

Conjunto de técnicas que hacen transparentes y comprensibles las decisiones de los modelos de IA.

Causal Inference in AI
